# Expense-Tracker-Python

## Description
This is a simple command-line Expense Tracker application built using Python. It allows users to add, view, and calculate total expenses. The data is stored in a CSV file.

---

## Features
- Add expense (description and amount)
- View all expenses
- Calculate total expense
- Data stored in CSV file

---

## Technologies Used
- Python
- CSV File Handling
- OS Module
